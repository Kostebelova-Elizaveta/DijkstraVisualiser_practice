# Dijkstra Visualiser

This program allows user to see how Dijkstra algorithm works

## Getting Started

### Dependencies

* Java 17/18
* OpenJFX (For Linux)

### Executing

* Run ```mvn package```
* Go to ```./target```
* Run ```java -jar StudyPract-*.*.jar```

### Testing

Run ```mvn test``` from root of the project.
